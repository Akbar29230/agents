# Как задеплоить с бэкендом на Vercel

## Структура (ВАЖНО: замени всё содержимое репозитория целиком на это)

```
твой-репозиторий/
├── api/
│   └── chat.js          ← бэкенд-прокси
├── src/
│   ├── App.jsx           ← твои агенты
│   ├── main.jsx           ← точка входа React
│   └── index.css          ← стили Tailwind
├── index.html             ← HTML-страница
├── package.json           ← список зависимостей
├── vite.config.js         ← настройка сборки
├── tailwind.config.js
└── postcss.config.js
```

Раньше в репозитории лежали только `App.jsx` и `api/chat.js` — этого недостаточно. Vercel не может собрать сайт без `package.json`, `index.html` и точки входа. Теперь весь "скелет" проекта на месте.

## Шаги

1. **Полностью замени содержимое репозитория** на эти файлы (удали старые `App.jsx`, `README-deploy.md` из корня — теперь `App.jsx` лежит внутри `src/`)

2. **Закоммить и запушь в GitHub**
   ```
   git add .
   git commit -m "Добавлен бэкенд-прокси для Claude API"
   git push
   ```

3. **Подключи репозиторий к Vercel**
   - Зайди на vercel.com → New Project → выбери свой GitHub-репозиторий
   - Framework Preset определится автоматически (Vite/CRA/Next — смотря что у тебя)

4. **Добавь переменную окружения**
   - В настройках проекта на Vercel: **Settings → Environment Variables**
   - Добавь: `ANTHROPIC_API_KEY` = твой настоящий ключ с console.anthropic.com
   - ⚠️ Никогда не клади ключ прямо в код или в GitHub — только через переменные окружения

5. **Deploy**
   - Vercel задеплоит автоматически при пуше. Или вручную: `vercel --prod`

## Если проект уже был создан на Vercel

Ничего пересоздавать не нужно — просто запушь новые файлы в тот же GitHub-репозиторий. Vercel сам увидит изменения и передеплоит проект автоматически. Если этого не произошло — зайди в Vercel → твой проект → Deployments → у последнего деплоя нажми **⋯ → Redeploy**.

## Проверка

После деплоя открой `https://твой-проект.vercel.app` — агенты должны работать, запросы теперь идут через твой сервер, ключ нигде не светится в браузере.

Если получишь ошибку — открой консоль браузера (F12), там будет текст ошибки от `/api/chat`, обычно это подскажет, не забыл ли добавить `ANTHROPIC_API_KEY` в Vercel.
